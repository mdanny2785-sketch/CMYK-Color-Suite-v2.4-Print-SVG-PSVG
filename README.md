# CMYK-Color-Suite-v2.4-Print-SVG-PSVG
CMYK Color Suite brings a print-safe SVG workflow to Inkscape with PSVG v1.0. Supports CMYK, spot colors, PDF/X export, preflight, and CLI tools. Built with help from Claude AI and ChatGPT. Looking for testers to validate real-world print workflows and help improve reliability.
